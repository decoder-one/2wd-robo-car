# 2wd-robo-car
2 dc motors, arduino uno with shield 5.0, L298N dual full-bridge driver , servo and ultrasonic sensor

complete set:
https://www.roboter-bausatz.de/p/bausatz-2wd-roboter-smart-car-arduino-kit
other kits see www

parts:
UNO R3 board
2 wheels with tires,
2 gear motors,
4 motor cables,
Acrylic chassis,
Motor driver,
Ultrasonic sensor with mount,
Servo motor,
Battery holder,
USB cable,
Assorted screws and nuts for assembly

!!! It is necessary to use two 18650 batteries for the power supply.
